# E-Prayog - Immersive Science Learning Platform

A premium virtual science learning platform for Karnataka PUC 1 & 2 students, featuring interactive 3D experiments, AI-powered tutoring, and board-aligned curriculum.

## 🚀 Features

- **40+ Virtual Experiments** - Interactive 3D simulations for Physics, Chemistry, Biology, and Mathematics
- **YOG AI Tutor** - Personalized learning assistance with visual explanations in Kannada and English
- **Board-Aligned Curriculum** - Complete coverage of Karnataka PUC syllabus
- **Real-time Dashboard** - Track progress, study time, and performance analytics
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices

## 🛠 Tech Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS with custom design tokens
- **Animations**: Framer Motion for UI animations
- **3D Graphics**: Three.js + React Three Fiber + Drei
- **Charts**: Recharts for data visualization
- **Routing**: React Router DOM

## 📦 Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd e-prayog
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## 🎯 Project Structure

```
src/
├── components/
│   ├── ui/           # Reusable UI components
│   ├── three/        # Three.js 3D components
│   └── [sections]/   # Page sections
├── hooks/            # Custom React hooks
├── data/             # Static data files
└── styles/           # Global styles and CSS variables
```

## 🎨 Design System

- **Colors**: Navy blue base with green and cyan accents
- **Typography**: Fraunces (display) + Plus Jakarta Sans (body)
- **Animations**: Smooth micro-interactions and scroll-based reveals
- **Components**: Glass morphism effects with subtle borders
